Quick Reference Guide

Key Financial Formulas

Present Value

PV = FV / (1 + r)^n

Where: FV = Future Value, r = interest rate, n = periods

Future Value

FV = PV √ó (1 + r)^n

Where: PV = Present Value, r = interest rate, n = periods

Loan Payment (PMT)

PMT = PV √ó [r(1+r)^n] / [(1+r)^n - 1]

Where: PV = loan amount, r = monthly rate, n = payments

Financial Ratio Formulas

Liquidity Ratios

Current Ratio = Current Assets / Current Liabilities

Quick Ratio = (Current Assets - Inventory) / Current Liabilities

Profitability Ratios

ROA = Net Income / Total Assets

ROE = Net Income / Shareholders' Equity

Profit Margin = Net Income / Revenue

Leverage Ratios

Debt-to-Equity = Total Debt / Total Equity

Times Interest Earned = EBIT / Interest Expense

Fundamental Accounting Principles

Basic Accounting Equation

Assets = Liabilities + Equity

This equation must always balance

Debit and Credit Rules

Debits increase: Assets, Expenses, Dividends

Credits increase: Liabilities, Equity, Revenue
